# Machine Learning Notebooks

This repository contains a collection of machine learning notebooks developed as part of my coursework and personal experiments. Each notebook focuses on a different dataset and learning technique, covering regression, classification, clustering, dimensionality reduction, and exploratory data analysis.

---

## üìö Contents

### 1. **Boston House Price Prediction**
**File:** `boston-house-price-prediction.ipynb`  
- **Task:** Predict median house prices in Boston suburbs.  
- **Techniques:** Linear Regression, Feature Scaling, Evaluation Metrics (MAE, RMSE).  
- **Dataset:** [Boston Housing Dataset (Scikit-learn)](https://scikit-learn.org/stable/datasets/toy_dataset.html#boston-dataset).  

---

### 2. **Breast Cancer Classification**
**File:** `breast-cancer-classification.ipynb`  
- **Task:** Classify tumors as malignant or benign.  
- **Techniques:** Logistic Regression, Random Forest, Model Evaluation (Confusion Matrix, ROC Curve).  
- **Dataset:** [Breast Cancer Wisconsin Dataset (Scikit-learn)](https://scikit-learn.org/stable/datasets/toy_dataset.html#breast-cancer-dataset).  

---

### 3. **Flower Classification**
**File:** `flower-classifiication.ipynb`  
- **Task:** Classify iris flowers into species.  
- **Techniques:** K-Nearest Neighbors (KNN), Decision Trees, Accuracy Comparison.  
- **Dataset:** [Iris Dataset (Scikit-learn)](https://scikit-learn.org/stable/datasets/toy_dataset.html#iris-plants-dataset).  

---

### 4. **Mall Customers Clustering**
**File:** `mall-customers-clustering.ipynb`  
- **Task:** Segment customers based on spending behavior.  
- **Techniques:** K-Means Clustering, Elbow Method, Cluster Visualization.  
- **Dataset:** [Mall Customers Dataset (Kaggle)](https://www.kaggle.com/datasets/shwetabh123/mall-customers).  

---

### 5. **Student Depression Prediction**
**File:** `student-depression-prediction.ipynb`  
- **Task:** Predict depression levels among students based on lifestyle and academic factors.  
- **Techniques:** Data Preprocessing, Feature Importance, Classification Models.  
- **Dataset:** Public academic dataset (sources vary by institution).  

---

### 6. **Titanic Dataset Exploration**
**File:** `titanic-dataset-exploration.ipynb`  
- **Task:** Analyze passenger data to predict survival.  
- **Techniques:** Data Cleaning, Feature Engineering, Logistic Regression, Random Forest.  
- **Dataset:** [Titanic Dataset (Kaggle)](https://www.kaggle.com/c/titanic).  

---

### 7. **Wine Dataset Dimensionality Reduction**
**File:** `wine-dataset-dimensionality-reduction.ipynb`  
- **Task:** Reduce feature dimensions for visualization and classification.  
- **Techniques:** PCA (Principal Component Analysis)
- **Dataset:** [Wine Dataset (UCI Machine Learning Repository)](https://archive.ics.uci.edu/ml/datasets/wine).
